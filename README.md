tsplit
======

This script define simple functions that allow you to make quick tmux interfaces.

**For example :** You can split a tmux session in 4 panels (2 by 2) with 
```
t4panes session0
```

Installation
------------

```bash
git clone https://github.com/SylvainDaste/tsplit.git
```
Then 
```bash
source ./tsplit.sh
```
